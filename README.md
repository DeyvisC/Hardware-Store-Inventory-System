# Hardware Store Inventory System 🛠️

A desktop application designed to streamline inventory control and sales processes. Built with **Java SE** following the **MVC (Model-View-Controller)** design pattern.

## 🚀 Key Features
* **Inventory Tracking:** Real-time stock updates.
* **Sales Management:** Automated inventory deduction.
* **MVC Architecture:** Clear separation of concerns for easy maintenance.

## 🛠️ Tech Stack
* **Language:** Java SE
* **GUI:** Java Swing
* **IDE:** Apache NetBeans
